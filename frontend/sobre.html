Sobre

Bem-vindo à Taverna do Animes Room, eu me chamo Gabriel, um dev apaixonado pela cultura japonesa e esse meu projeto pessoal é o ponto de encontro oficial para quem busca conhecimento. Se você está procurando um aliado leal para futuras missões, deseja recrutar um desenvolvedor para sua guilda ou apenas quer trocar uma ideia sobre o vasto mundo do desenvolvimento web, sinta-se em casa! Nada melhor que uma troca de conhecimento, certo? Me chame nas redes, e bora trocar uma ideia!

Projeto

Este site nasceu como um projeto totalmente sem fins lucrativos, focado exclusivamente no meu desenvolvimento como programador. Decidi unir o útil ao agradável: criar um espaço dedicado à cultura otaku enquanto forjo minhas habilidades técnicas em HTML5, CSS3 e JS.
O Animes Room é o meu laboratório pessoal e uma peça fundamental do meu portfólio, onde cada linha de código representa um novo nível de experiência que conquistei nesta jornada!

Stack

Para dar vida ao Animes Room, não usei atalhos ou construtores automáticos. Este artefato foi forjado com Front-End puro, utilizando a base sólida do HTML5 e a estilização precisa do CSS3 para garantir que cada pixel esteja no lugar certo. Mas a verdadeira magia acontece nos bastidores: o motor deste projeto roda em Node.js com Express.js, garantindo uma comunicação rápida e eficiente.
Para guardar todos os registros e dados dos nossos heróis, utilizei o poder do MySQL, um banco de dados robusto que mantém nossa biblioteca de animes sempre organizada e segura. É o Full-Stack na prática, do design ao banco de dados!
