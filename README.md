# Animes Room

O **Animes Room** é um projeto web desenvolvido para fins de estudo e portfólio.

A aplicação simula uma plataforma de animes onde é possível visualizar banners, títulos, informações sobre os animes e listar episódios por temporada.

O principal objetivo do projeto é praticar **desenvolvimento Full Stack**, integrando **frontend, backend e banco de dados**.

---

## Tecnologias Utilizadas

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Banco de dados:
- MySQL

---

## Funcionalidades

- Exibição de banners e títulos de animes
- Listagem de episódios por temporada
- Carregamento dinâmico de dados utilizando **fetch API**
- API backend para busca de episódios
- Integração com banco de dados MySQL
